## यह क्या है?

"एस्केप द वैली" एक सर्वाइवल गेम है जो "ओरेगन ट्रेल" की शैली पर आधारित है और यह आपके कंप्यूटर टर्मिनल पर चलता है। आप बसुकों के एक समूह का नेतृत्व करें और एक ऐसी जंगली भूमि को पार करें जो हर बार अलग होती है। भोजन, पानी, गाड़ी की स्थिति और मनोबल का प्रबंधन करें, और साथ ही घटनाओं, खतरों और मुश्किल विकल्पों का सामना करें।

एक वैकल्पिक एआई गेम मास्टर (जो ओलामा द्वारा संचालित है), तीन अलग-अलग कहानी कहने की शैलियों में आपकी यात्रा का वर्णन करता है। एक वैकल्पिक एक्सआरपीएल टेस्टनेट लेजर बैकपैक आपके आपूर्ति में होने वाले परिवर्तनों को रिकॉर्ड करता है, जो "ऑन-चेन रसीद" के रूप में काम करता है - यह प्रमाण है कि आप जीवित रहे, या यह प्रमाण है कि आपने प्रयास किया।

## शुरुआत कैसे करें।

```bash
pip install -e ".[dev]"

# Launch the full-screen TUI (recommended)
trail tui --seed 42

# Resume a saved game
trail tui --continue

# With AI narration (requires Ollama running locally)
trail tui --seed 42 --voice

# Without AI narration (deterministic mode)
trail tui --seed 42 --gm-off
```

## खेलने का तरीका।

प्रत्येक बारी में, आप शिविर से एक कार्य का चयन करते हैं:

| कार्रवाई। | यह क्या करता है। |
|--------|-------------|
| **Travel** | घाटी के मुहाने की ओर बढ़ें। भोजन और पानी की व्यवस्था के लिए खर्च आएगा। मशीन खराब होने और अन्य अप्रत्याशित घटनाओं का खतरा है। |
| **Rest** | पार्टी को एकजुट करें, मनोबल बढ़ाएं। इसमें संसाधनों की लागत आती है, लेकिन कोई प्रगति नहीं होती। |
| **Hunt** | गोली खर्च करें, ताकि भोजन मिलने की संभावना बढ़ जाए। यह जंगलों और समतल इलाकों में अधिक प्रभावी होता है। |
| **Repair** | गाड़ी को ठीक करने के लिए एक अतिरिक्त पुर्जे का उपयोग करें। यह जीवित रहने के लिए बहुत महत्वपूर्ण है। |

**घटनाएं** यात्रा में बाधाएं उत्पन्न करती हैं और विकल्पों (ए/बी/सी) के साथ प्रस्तुत होती हैं। सतर्क विकल्प अधिक सुरक्षित होते हैं, लेकिन इसमें समय लगता है। साहसिक विकल्प तेज़ होते हैं, लेकिन जोखिम भरे होते हैं। कोई भी विकल्प हमेशा सही नहीं होता।

**यह गाड़ी सब कुछ है।** यदि यह खराब हो जाती है और इसके पुर्जे उपलब्ध नहीं हैं, तो यात्रा समाप्त हो जाएगी। इसकी स्थिति को आधा से अधिक बनाए रखें और अस्थायी रूप से खराबी से बचने के लिए, रखरखाव के लिए समय निकालें (थोड़ा आराम करें, फिर मरम्मत करें)।

**गति:** यह गति और सुरक्षा के बीच संतुलन बनाए रखता है। सामान्य गति (डिफ़ॉल्ट) सबसे सुरक्षित होती है। तेज़ गति से अधिक दूरी तय की जा सकती है, लेकिन इससे संसाधनों की खपत अधिक होती है और गाड़ियों के खराब होने की संभावना बढ़ जाती है।

**आपातकालीन निकास वाल्व** (जैसे कि सख्त राशन, हताश मरम्मत, कार्गो को छोड़ देना) आपातकालीन स्थितियों के लिए मौजूद होते हैं। इनके कुछ नकारात्मक प्रभाव और "कूलडाउन" (समय अंतराल) होते हैं - ये अंतिम उपाय हैं, कोई रणनीति नहीं।

अधिक विस्तृत जानकारी के लिए, [उत्तरजीविता गाइड](docs/survival-guide.md) देखें।

## जीएम प्रोफाइल।

एआई (कृत्रिम बुद्धिमत्ता) वाला कथावाचक कहानी के लहजे को बदलता है, लेकिन खेल के नियमों को नहीं। तीनों प्रोफाइल एक ही गेम खेलते हैं।

- **क्रॉनिकलर (Chronicler)** — यथार्थवादी, व्यावहारिक, और संक्षिप्त। इसमें लोककथाओं का बहुत कम उपयोग है। यह केवल उन घटनाओं की रिपोर्ट करता है जो हुई हैं।
- **फायरसाइड (Fireside)** — एक गंभीर, आग के किनारे बैठकर कहानियाँ सुनाने वाला व्यक्ति। इसमें सूक्ष्म, अजीबोगरीब क्षण होते हैं। यह डिफ़ॉल्ट विकल्प है।
- **लैंटर्न-बेयरर (Lantern-Bearer)** — यह अजीब और रहस्यमय है, लेकिन फिर भी वास्तविकता से जुड़ा हुआ है और इसके परिणाम होते हैं। यह सबसे विचित्र चरित्र है।

`--gm-profile` विकल्प के साथ कमांड: `trail tui --gm-profile lantern`

## लेजर बैकपैक (वैकल्पिक)

"लेजर बैकपैक" आपकी 5 महत्वपूर्ण आवश्यकताओं (भोजन, पानी, दवाइयां, गोला-बारूद, पुर्जे) को एक्सआरपीएल टेस्टनेट पर टोकन के रूप में ट्रैक करता है। प्रत्येक शहर के चेकपॉइंट पर, एक लेनदेन रिकॉर्ड "ऑन-चेन" दर्ज किया जाता है। आपकी यात्रा के अंत में, आपके "ट्रेल लेजर" में लेनदेन आईडी शामिल होंगे, जिन्हें कोई भी सत्यापित कर सकता है।

यह सुविधा पूरी तरह से वैकल्पिक है। गेम बिना इस सुविधा के भी बिल्कुल उसी तरह चलता है (यह डिफ़ॉल्ट सेटिंग है)। आप इसे टीयूआई (टेक्स्ट-आधारित यूजर इंटरफेस) में मौजूद 'एल' मेनू से या कमांड लाइन इंटरफेस (सीएलआई) के माध्यम से सक्रिय कर सकते हैं:

```bash
trail ledger enable
trail ledger status
trail ledger reconcile  # retry failed settlements
```

इसके लिए `xrpl-py` नामक निर्भरता स्थापित करने के लिए `pip install -e ".[xrpl]"` कमांड का उपयोग करना आवश्यक है।

## कमांड्स (आदेश)

| आदेश। | विवरण। |
|---------|-------------|
| `trail tui` | पूर्ण-स्क्रीन टेक्स्ट आधारित यूजर इंटरफेस शुरू करें। |
| `trail new` | एक नया रन शुरू करें (क्लासिक कमांड-लाइन मोड)। |
| `trail play` | एक सहेजे गए सत्र को जारी रखें (क्लासिक कमांड-लाइन इंटरफेस मोड)। |
| `trail status` | शो, वैगन और आवश्यक सामग्री दिखाएं। |
| `trail journal` | नवीनतम जर्नल प्रविष्टियाँ दिखाएँ। |
| `trail self-check` | खेल के वातावरण की स्थिति की जांच करें। |
| `trail version` | संस्करण दिखाएँ। |
| `trail ledger status` | बैकपैक की स्थिति दिखाएं। |
| `trail ledger enable` | XRPL बैकपैक को सक्रिय करें। |
| `trail ledger disable` | XRPL बैकपैक को निष्क्रिय करें। |
| `trail ledger settle` | एक चेकपॉइंट को मैन्युअल रूप से सेट करें। |
| `trail ledger reconcile` | विफल हुए लेनदेन को फिर से करने का प्रयास करें। |
| `trail ledger wallet` | वॉलेट की जानकारी दिखाएं। |
| `trail parcel list` | प्राप्त हुए पार्सल की सूची। |
| `trail parcel accept <id>` | किसी लंबित पार्सल को स्वीकार करें। |

## चेतावनी संदेश।

डिफ़ॉल्ट रूप से, यह गेम नए खिलाड़ियों को खतरों को जल्दी पहचानने में मदद करने के लिए विस्तृत चेतावनी संदेश दिखाता है। अनुभवी खिलाड़ी "न्यूनतम मोड" में स्विच कर सकते हैं, जो केवल अंतिम क्षण में आने वाले गंभीर खतरों की चेतावनी देता है।

```bash
trail tui --callouts minimal
trail new --callouts minimal
```
