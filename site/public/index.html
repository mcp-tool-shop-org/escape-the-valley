01

What's new on the trail

v1.1 — the highlights

01 · NARRATION

Streaming AI narration

Three GM profiles — Chronicler, Fireside, Lantern-Bearer — narrate in distinct voices, now streamed token-by-token. Runs locally on Ollama, zero API cost. The GM never bricks a run: if the model's down, the trail tells its own story.

02 · ENGINE

The engine decides

Seeded, deterministic, reproducible. The GM tells the story; the engine is the rules-lawyer. Same seed plus the same choices yields the same run, every time.

03 · STAKES

Real stakes, graded endings

Events can wound and kill; deaths are attributed — injury, dysentery, exposure. Runs end triumphant, weathered, pyrrhic, or lost, with a narrated epilogue read from who survived, how long it took, and what you sacrificed.

04 · LEDGER

The ledger is the receipt book

Optional on-chain inventory on the XRP Ledger testnet: supply tokens, settlement receipts, parcels you trade with other travelers. A reconciliation proof verifies the on-chain memos themselves — the engine cannot fake the ledger.

05 · TERMINAL

A terminal that never freezes

A full-screen Textual TUI with async narration ("the storyteller is thinking…"), reachable last-resort escape valves, an end-of-run postcard, and honest signals when the AI or network is degraded.

06 · HARDENED

Hardened

685 tests, 0 critical / 0 high, crash-safe atomic saves, testnet-only-in-code. Forged by a full dogfood-swarm hardening pass.

02

Break camp in three commands

Python 3.11+ · Ollama optional · XRPL optional

Install
$ pip install escape-the-valley
# with on-chain ledger
$ pip install "escape-the-valley[xrpl]"
# from source, for hacking
$ pip install -e ".[dev,xrpl]"
Play
# launch the full-screen TUI
$ trail tui --seed 42
# with AI narration aloud
$ trail tui --seed 42 --voice
# deterministic, GM off
$ trail tui --gm-off
Commands
$ trail tui --continue
$ trail status
$ trail journal -n 5
$ trail ledger enable
$ trail self-check
03

Choose your narrator

The narrator shapes the tone, not the mechanics. All three play the same game.

Chronicler

--gm-profile chronicler

Grounded, practical, spare. Minimal folklore — reports what happened. For players who want facts.

Fireside

DEFAULT

Serious campfire narrator. Subtle uncanny moments. The place to start your first playthrough.

Lantern-Bearer

--gm-profile lantern

Uncanny and liminal, but still grounded in consequence. The weird one — for experienced travelers.
